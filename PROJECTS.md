## 🌟 Featured Projects

### 🧠 NovaAI

A real-time, voice-driven AI companion built to run fully local — no cloud required, no artificial limits, just your own assistant on your own terms.
**Stack:** Python
**Repo:** https://github.com/cachenetworks/NovaAI

### ☁️ NovaDrive

A Google Drive-inspired web app that uses **Discord or S3-compatible object storage** as the backend.
It handles chunked uploads, manifest storage in SQLite, verification with SHA256, sharing, WebDAV access, and a polished startup-style UI.
**Stack:** Python, Flask, SQLite, Tailwind/Jinja
**Repo:** https://github.com/cachenetworks/NovaDrive

### 💬 DollarCord

A real-time Discord-inspired chat platform with servers, channels, DMs, presence, typing indicators, reactions, pins, and settings.
**Stack:** Next.js, Prisma, SQLite, Socket.IO, Tailwind CSS
**Repo:** https://github.com/cachenetworks/DollarCord

### 🔐 CacheSec

A Raspberry Pi 5 face-recognition security system with **Kinect v1 support**, night vision, Discord alerts, and a Flask admin UI.
Because normal security systems apparently weren’t interesting enough.
**Stack:** Python, Flask, Raspberry Pi
**Repo:** https://github.com/cachenetworks/CacheSec

## 🛠️ Systems, Tools, and Experiments

### 🧱 FurOS

A minimal **x86_64 freestanding kernel** that boots with GRUB in BIOS/Legacy mode and runs a terminal-only shell in VGA text mode.
Includes filesystem commands, a simple editor, disk tools, install flow, and even a simulated package manager.
**Stack:** C, Assembly, Makefile
**Repo:** https://github.com/cachenetworks/FurOS

### 🌐 CxrruptTranslate

A translation tool built for **VRChat**, with real-time translation, OSC integration, language auto-detection, and a purple/black UI.
Basically: less language barrier, more social chaos.
**Stack:** Python, PyQt6
**Repo:** https://github.com/cachenetworks/CxrruptTranslate

### 🔊 CxrruptPad

A modern audio soundboard project with tags including open-source, Discord, VRChat, and soundboard tooling.
**Stack:** Python
**Repo:** https://github.com/cachenetworks/CxrruptPad

### 🗣️ CxrruptTTS

A text-to-speech project currently public on your profile, though it does not currently expose a public short description on the repo page.
**Repo:** https://github.com/cachenetworks/CxrruptTTS
